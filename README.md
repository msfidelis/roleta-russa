# Roleta russa do Bash
Roleta russa do bash. Jogue em containers, ou nos servers de produção. Cada um com suas prioridades.


## Safe Mode

### Jogue em containers

#### Build

```bash
docker build -t roleta .
```

```bash
docker run -it roleta
```

#### Ooou

```bash
 docker run -it msfidelis/roleta-russa
```
